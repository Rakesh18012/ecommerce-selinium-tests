# Ecommerce Website Selenium Automation

Author: Rakesh Kurma

This is a UI automation project for a demo ecommerce website created for hands-on practice and showcasing QA skills.

## Tech Stack
- Selenium WebDriver
- TestNG
- Maven
- Java

## Features Automated
- User login
- Product search
- Add to cart
- Checkout flow
